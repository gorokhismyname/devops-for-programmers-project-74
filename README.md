# Project 1: Docker compose app

This is a simple training app for learning docker compose main functions and basic ci principles

## Requirements

* docker
* docker compose V2
* make

## Usage

### Setup app

```bash
make compose-setup
```

### Run tests

```bash
make compose-test
```

### Start application

```bash
make compose-start
# open http://0.0.0.0:8080 in browser
```
